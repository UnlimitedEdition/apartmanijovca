# Luxury Booking Page Redesign - Requirements

## Problem
Trenutna booking stranica izgleda "trulo" i "poor" - nije dovoljno atraktivna i profesionalna za premium smeštaj.

## Cilj
Kreirati luksuznu, premium booking stranicu koja:
1. Izgleda moderno i skupo
2. Koristi gradijente i animacije
3. Ima premium feel
4. Zadržava svu funkcionalnost
5. Radi na svim uređajima

## Dizajn Elementi

### Hero Sekcija
- Gradient pozadina (blue → indigo → purple)
- Pattern overlay za teksturu
- Wave divider na dnu
- Premium badge
- Veliki, bold naslov

### Progress Indicator
- Bela kartica sa senkom
- Gradient progress bar
- Velike, zaobljene ikone
- Animacije za trenutni korak
- Checkmark za završene korake

### Step Kartice

#### Step 1: Datumi
- Gradient header (blue → indigo → purple)
- Ikona u headeru
- Premium summary kartica sa:
  - 4 kolone sa ikonama
  - Gradient ikone (različite boje)
  - Velike brojke
  - Senke
- Gradient dugme

#### Step 2: Opcije
- Isti stil kao Step 1
- Checkbox kartice sa hover efektima
- Textarea sa zaobljenim uglovima

#### Step 3: Detalji
- Isti stil
- Premium input polja
- Terms checkbox sa plavom karticom
- Gradient dugme

### Success Stranica
- Gradient pozadina (emerald → teal → cyan)
- Animirana success ikona
- Bounce animacija
- Premium kartice za summary
- Instrukcije sa numerisanim koracima
- Gradient ikone

### Boje
- Primary: Blue (600) → Indigo (600) → Purple (600)
- Success: Emerald (400) → Teal (500)
- Warning: Amber (500) → Orange (500)
- Error: Red (500) → Rose (500)

### Animacije
- Hover scale (1.02)
- Active scale (0.95)
- Pulse za trenutni korak
- Bounce za success
- Smooth transitions (300ms)

### Senke
- shadow-2xl za kartice
- shadow-lg za ikone
- Colored shadows (blue-500/30, emerald-500/50, itd.)

## Funkcionalnost
- Sve postojeće funkcije ostaju
- Validacija ostaje
- API calls ostaju
- Translations ostaju
- Accessibility ostaje

## Responsive
- Mobile-first pristup
- Breakpoints: sm, md, lg, xl
- Stack na mobilnom
- Grid na desktopu
